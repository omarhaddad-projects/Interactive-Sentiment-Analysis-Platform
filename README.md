# Interactive-Sentiment-Analysis-Platform

A Colab-ready hybrid sentiment analysis demo for single-text inference, transparent keyword inspection, TF-IDF feature interpretation, and batch CSV prediction workflows.

## Repository contents
- `Interactive-Sentiment-Analysis-Platform.ipynb`
- `data/sample_texts.csv`: sample file for the batch workflow.
- `assets/sample_result.png`: repository preview image.
- `assets/hybrid_scores.png`: sentiment score plot.
- `assets/feature_contributions.png`: feature contribution plot.
- `requirements.txt`: dependency list aligned with the Colab notebook.

## Main capabilities
- interactive sentiment analysis
- keyword inspection
- feature contribution exploration
- batch CSV inference
- Colab-oriented notebook execution

## Setup
Open the notebook in Google Colab and run it in a fresh runtime. The notebook installs its own dependencies and downloads the external resources used by the hybrid sentiment pipeline.

## Usage
Use the single-text section for direct inference and the batch section for CSV-based prediction.
